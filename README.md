# applet-taro-cli

Taro + webpack + nutui/taro-ui +vue3/react 的脚手架

## 项目目录结构

```
├── .husky/             // git hook 工具配置
├── .vscode/            // vscode 代码格式化规范
├── config/             // 项目构建配置
├── env/                // 项目环境变量配置
├── scripts/            // 预执行工具
└── src/
    ├── ajax/               // ajax 二次封装
    ├── api/                // api 接口
    ├── assets/             // 静态资源目录
    ├── components/                 // 公共组件目录
    ├── hooks/                 // 项目hook目录
    ├── style/              // 通用 CSS 目录
    ├── utils/              // 工具函数目录
    ├── pages/              // 页面组件目录
    ├── app.config.ts
    ├── app.scss
    ├── app.ts
    ├── index.html
├── .cz-config.js                  // 单元测试目录
├── .editorconfig
├── .eslintrc.js
├── .gitignore
├── .prettierrc
├── babel.config.js
├── commitlint.config.js
├── global.d.ts
├── project.config.json
├── project.tt.json
├── README.md
├── tsconfig.json           // TypeScript 配置文件
├── package-lock.json
└── package.json
```
